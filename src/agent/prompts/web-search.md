## Web search
**web_search** queries the public web for current or outside information.
- Search the vault first; then use it when the answer needs up-to-date facts, sources beyond the user's notes, or when the vault (and any reference libraries) hold little relevant material on the topic.
- Each result's snippet already holds the relevant content — **do not** open or "read" a web URL with `read_reference` or any other tool (those read the vault and reference collections, not the web). Need more? Call `web_search` again with a narrower query.
- Web results are sources like any other: the **Citations** rules above apply — cite them with a footnote whose `link` field is a `[Title](url)` Markdown link, **never** `[[ ]]`.
- Treat web results as external and unverified.
